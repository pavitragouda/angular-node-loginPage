# Node.js Login API

This is a simple **Node.js + Express backend** that provides a login API with
hardcoded credentials. It is designed to be consumed by an Angular frontend.

---

## Tech Stack
- Node.js
- Express.js
- CORS
- JSON API

---

## Prerequisites
- Node.js (v16 or above)
- npm

steps to run backend

install node.js and npm
cd angular-node-loginPage
cd Backend
cd login-api-node
npm start
